Add a License to a Project

Note

If you just bought a license and want to apply your license to a project that is showing a warning banner at the top that upgrades are needed, then this section is for you.

To apply a license to a project:

  1. Open the project.

  2. In the left Activity Bar, click Upgrades to open the Upgrades panel.

  1. Scroll down to the Licenses section. You will see the following:

  1. Click the button for Upgrade using a license key….

  2. Click in the empty area that says Enter license code or select from the <n> licenses that you manage. If you have any licenses, you will see a list of available licenses that you can apply.

  3. Click to select the license you want to apply to the current project. Doing this will enable the blue Apply license button.

  4. Click Apply license.

  5. Restart the project.

You can repeat the above steps to add more licenses to the current project.

If the license is not successfully applied, see License error messages.
